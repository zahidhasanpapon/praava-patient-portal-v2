## Environment Variables

Create a new file .env:

```sh
touch .env
```

Required fileds so far:

- NODE_ENV
- PORT
- PG-USER
- PG_PASSWORD
- PG_HOST
- PG_PORT
- PG_DATABASE
- JWT_SECRET_ACCESS

## Getting Started

Clone the repo:

```sh
git clone https://github.com/zahidhasanpapon/server-developer-praava.git
cd server-developer-praava
```

Install dependencies:

```js
npm install
```

Start the server:

```sh
npm start
```
